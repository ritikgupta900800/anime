Add your wallpapers

  1. Grab direct image links (e.g., from your own hosting, Google Drive direct links, or any CDN you control).
  2. Edit the SAMPLE_WALLPAPERS list in this file — add title, series, tags, url, and res (like 3840x2160).
  3. Save as index.html. Host on GitHub Pages, Netlify, or Vercel.
  4. Copyright tip: Only upload images you own or have permission to share.
